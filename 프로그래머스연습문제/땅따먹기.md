#### 연습문제

1. 첫번째 

   테스트케이스만 성공하고 다 틀림..

   ```python
   def solution(land):
       answer = 0
       cnt=-1
       for i in land:
           max=0
           for j in range(len(i)):
               if max<i[j] and cnt!=j:
                   max=i[j]
           else:
               cnt=i.index(max)
           answer +=max
       return answer
   ```
