1. 38% 시간초과 

```python
import sys
n=int(sys.stdin.readline())
arr=list(map(int,sys.stdin.readline().split()))
stack=[]
for i in range(len(arr)):
    for j in range(i+1,len(arr)):
        if arr[i]<arr[j]:
            stack.append(arr[j])
            break
    else:
        stack.append(-1)
for i in stack:
    print(i,end=' ')
```
