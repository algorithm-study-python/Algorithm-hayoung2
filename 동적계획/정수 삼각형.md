##  정수 삼각형

#### 동적계획법

1. 첫번째 예제 맞고, 7.1 맞음..

```python
def solution(triangle):
    tmp=[triangle[0][0]]
    cnt=1
    for numbers in triangle[1:] :
        result=[]
        for j in range(len(triangle[cnt-1])) :
            result.append(tmp[j]+triangle[cnt][j])
            result.append(tmp[j]+triangle[cnt][j+1])
        cnt +=1 
        tmp=result
            
    return max(tmp)   
```
